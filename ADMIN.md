# Admin Access (Kalakaar)

## Admin URL
- `/admin/login.php`

## Default Login (after importing `database.sql`)
- Username: `admin`
- Password: `password`

## Change Password
- Go to `/admin/password.php`
- Enter:
  - Username or Email: `admin`
  - Current password: `password`
  - New password: (your new password)

## Admin Dashboard
- `/admin/dashboard.php`

You can:
- View latest talent profile submissions
- View latest contact messages
- Update status for profiles and contacts

## Website Settings (Edit site info, theme, media)
- `/admin/settings.php`

You can edit:
- Header / brand name
- Homepage tagline & description
- Mobile number, email, address
- Social links (Instagram / Facebook / YouTube)
- Theme colors (primary/secondary)
- Upload Logo/DP, Hero images, About image

## Logout
- `/admin/logout.php`

## Important
- Change the default password immediately after first login.
- Keep `config/database.php` updated with your production database credentials.
